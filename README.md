# Hybrid CNN-BiLSTM for Real-Time Water Quality and Dissolved Oxygen Prediction in Aquaculture Systems

A hybrid deep learning framework using Convolutional Neural Networks (CNN) and Bidirectional Long Short-Term Memory (BiLSTM) for real-time water quality analysis and dissolved oxygen prediction in smart aquaculture systems.

## 📌 Overview
This project predicts dissolved oxygen (DO) levels using multiple water quality parameters such as pH, salinity, alkalinity, ammonia, and dissolved oxygen. The hybrid CNN-BiLSTM model captures both spatial and temporal relationships in the data to improve prediction accuracy.

## 🚀 Features
- Water quality data preprocessing and normalization
- CNN-based feature extraction
- BiLSTM-based temporal pattern learning
- Dissolved oxygen prediction
- Performance evaluation using RMSE and MAE
- Visualization of actual vs predicted values
- Error analysis and residual plots
